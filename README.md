### 环境需求

Node.js 8.x以上

### 开始构建

```shell
npm install
# 本地预览
npm run dev
# 构建生产环境
npm run build
# 推送到部署分支 dist
npm run deploy
# 建构生产环境并推送到部署分支
npm run buildAndDeploy
```
