使用案例

用户与案例

中国高校

快速入门

  1. 区域查询接口都在app/Application/Area/Model目录下，包含省份(Province)、城市(City)、区县(District)
  2. HTTP接口在app/Application/Area/Controller目录下

接口示例

通过关键字获取学校列表

注:keyword为学校名称关键字（示例：广东）

http://{你的站点}/ztbcms/index.php?g=Area&m=Api&a=getSchoolList&keyword=广东

返回数据：

{
  "status": "success",
  "data": [
    {
      "school_id": "1814",
      "school_name": "广东工业大学",
      "province_id": "440000",
      "type_id": "1"
    },{
      "school_id": "1816",
      "school_name": "广东海洋大学",
      "province_id": "440000",
      "type_id": "1"
  }
  ],
   "state": "success"
}

通过省份id获取学校列表

注:province_id 所属省份id

http://{你的站点}/ztbcms/index.php?g=Area&m=Api&a=getSchoolListByProvinceId&province_id=440000

返回数据：

{
  "status": "success",
  "data": [

      "type_id": "1"

}
