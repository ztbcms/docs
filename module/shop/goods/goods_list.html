# 商城-获取商品列表

# request

method ：get

URL： index.php?g=Shop&m=GoodsApi&a=add_cart

字段 类型 是否必须 描述
page int 否 当前页数
limit int 否 每页显示数量
is_recommend bool 否 是否推荐
is_new bool 否 是否新品
is_hot bool 否 是否热销
catid int 否 商品所属栏目，默认是全部
order string 列表的排序方式 eg : id desc
