# 用户-注册

# request

method ：post

URL： index.php?g=Shop&m=UserApi&a=register

字段 类型 是否必须 描述
username varchar 是 用户注册的邮箱或者手机
password varchar 是 用户注册密码
password2 varchar 是 用户注册确认密码
